# Quadcopter Drone (APM2.8)

Hands-on robotics project. Focused on hardware assembly, 
soldering, and flight controller calibration.

## Hardware
- APM2.8 flight controller
- 4x brushless motors + ESCs
- 1045 propellers
- 2S LiPo battery
- F450 frame

## What I Did
- Soldered power distribution board and ESC signal lines
- Assembled frame and mounted components
- Calibrated APM2.8 via Mission Planner
- Radio transmitter binding and basic flight tuning

## What I Learned
- Power distribution and LiPo safety
- ESC calibration and motor direction
- How flight controllers abstract PID into user-facing parameters
- Hardware debugging (cold solder joints, wrong motor order)
